# CS400NI – Programming Assessment

A collection of Java programming exercises completed as part of the CS400NI course, organised week by week.

## Repository Structure

```
CS400NI-Programming-Assessment/
├── Week 3/   – Java Basics & Operators
├── Week 4/   – Control Flow (if/else, switch)
├── Week 5/   – Loops (for, while, do-while)
├── Week 6/   – Arrays (1-D & 2-D)
└── Week 7/   – Object-Oriented Programming
```

## Weekly Breakdown

### Week 3 – Java Basics & Operators
| Exercise | Description |
|---|---|
| Data Type Inspector | Declares and prints all eight Java primitive types |
| Default Values | Demonstrates default values of instance variables |
| Grade Evaluator | Reads a grade and outputs Pass / Fail using a ternary operator |
| Literal Practice | Explores different literal forms in Java |
| Math Operations | Covers arithmetic, unary, assignment, relational, and boolean operators |
| Rickshaw Fare | Calculates a rickshaw fare with distance, time, local discount, and night surcharge |

### Week 4 – Control Flow
Nine lab exercises (`Lab1` – `Lab9`) practising `if`, `if-else`, nested `if`, and `switch` statements for problems such as pass/fail checks, divisibility tests, grade-to-GPA conversion, and more.

### Week 5 – Loops
Ten question files (`Q1` – `Q10`) covering `for`, `while`, and `do-while` loops, including a Nepal Investment Bank fixed-deposit calculator (`FixedDepositNIB`) that uses loop-based input validation.

### Week 6 – Arrays
Eight question files (`Q1` – `Q8`) covering 1-D array declaration and initialisation, traversal with loops, and 2-D arrays (e.g. a book catalogue storing categories, titles, and prices).

### Week 7 – Object-Oriented Programming
Seven question files (`Q1` – `Q7`) introducing classes, fields, constructors, and methods through real-world models:
| Exercise | Classes |
|---|---|
| Q1 | `Book`, `Main` |
| Q2 | `Rectangle`, `Q2` |
| Q3 | `Employee`, `Main` |
| Q4 | `Laptop`, `Main` |
| Q5 | `Mobile`, `Main` |
| Q6 | `Result`, `Main` |
| Q7 | `BankAccount`, `Main` |

## Prerequisites
- Java Development Kit (JDK) 8 or later
- Any Java IDE (BlueJ project files are included) or `javac` / `java` on the command line

## Running an Exercise
```bash
# From the exercise directory, e.g. Week 7/q7
javac BankAccount.java Main.java
java Main
```
